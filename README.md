<div>
	<img align="left" width="135" src='https://img3.doubanio.com/view/subject/s/public/s34392833.jpg'/><div>RATE: ★★★✰☆</div>
	<br/>
	<div>这本书讲述一个犹太男人与三位女人周旋的故事。有一位长期居住陪伴在男主身边的农妇女佣，善良淳朴，却也没有文化，目不识丁，无法与男主有精神上的共鸣。也许是因为这样，男主找寻了另一位所谓的“真爱”，以出差卖书的名义，总是与其幽会。也正是因为农妇生活简单，没有头脑，她总是没有意识到这一次次出差就是次次出轨。犹太人身处异国他乡，想方设法不让他人知道他有两个“老婆”。为此他从不泄露任何联系方式，不留电话，不留住址。而在犹太人本就戏剧化的生活里，又出现了一位不速之客——男主的原配正妻。没人能想到她在集中营身中数枪之后活了下来，但在大屠杀的背景下，离奇的事情并不少见。男主不得不在三个女人间周旋。农妇的救命之恩他必须回报，真爱的美貌和精神契合令他向往，念念不忘，以及突然冒出来的原配，他以为对她不再有感情，原配却为他坚守，从未委身于他人，这份心无疑让他有过动摇。他犹豫不决，无法取舍。在一系列闹剧之后，他终于决心与“真爱”私奔，甚至丢下几乎要生产的农妇。私奔的准备过程中，“真爱“的家被劫匪一掠而空，岳母也意外离世，一出出突发状况使得真爱的精神终究崩溃，于是男主也崩溃了，他本就无法承担起任何责任。最后，她选择了自杀，而犹太男人，选择了离开所有人，从此杳无音讯。</div>
	<br/>
	<div>作者把离奇的故事描绘得如此写实，就像是身边切实发生过的。文字是平和的叙述，不堆砌词藻，但里面的每个细节都恰到好处，展现了作者扎实的功底，也不乏平时对人类生活的细致观察。</div>
</div>

#### 4.[《林中之死》](https://book.douban.com/subject/35539397/) - 2025.05.07 [[美]舍伍德·安德森 Sherwood Anderson](https://www.douban.com/personage/27257642/)
<div>
	<img align="left" width="135" src='https://img9.doubanio.com/view/subject/s/public/s33950905.jpg'/><div>RATE: ★★★✰☆</div>
	<br/>
	<div>我会说，他的风格的确很明显，很独特。虽然很难想象那么多文学大师，都把他作为某种先驱来看待。但至少的确可以从故事里窥探一二。每个故事的叙述、过程，都简明扼要，言语精简，过渡自然。这些故事并不都那么令人印象深刻，不过我确实可以感受到它们统一的风格。风格是印象深刻的。</div>
	<br/>
	<div>每个短篇的故事都比较精简。我最喜欢的故事是《在陌生小镇》，讲述了一位教授的隐晦的婚外情，亦或是所谓的精神出轨。事实上，教授并没有实际的出轨行为，但仅仅是与学生独处在办公室，你就可以感受到暧昧的情愫在两人间悄然发生。即使故事里没有写明，却暗示学生是因为无法与教授在一起，才会失了神，与汽车相撞身亡。我想故事是这样引导着读者往这方向去想的，至少我是这样想的。而教授本人也许也感受到了，他当然感受到了，因此他怅然若失，为这一份潜在的新的感情而感伤。当这极小潜在的可能随着伊人逝去而彻底消失，仿佛也有什么东西从他心中失去了。是一份难得重燃的青春激情吗？很难描述这种情愫，我想它介乎于尊重与仰慕之间。于教授，比尊重更多一些，参杂一些爱慕怜惜，于学生，比仰慕更多一些，参杂一些可望不可得的痛苦。教授作为成年人做出了自己的选择，理智的、正确的、成熟的选择，他最终没有走出越界的那一步，却也无法逃避心中的真实情感，所以他去往一个陌生小镇，一个可以让他独处的地方，让他可以静静地哀悼或独自怀念的地方。
</div>
	<br/>
	<div>题外话，网络上或书店里有关舍伍德的书真是少之又少，偶然一次在西西弗书店找到《小城畸人》，迫不及待就买了，回去发现是纯英文。真是件愚蠢得令人印象颇深的事，不退是因为给自己买了一个物理上的教训，好让自己以后每次看到它，都能提醒自己不要为无谓的焦急付出任何金钱上的代价。于是又买了这本书。</div>
</div>

#### 5.[《乞力马扎罗的雪》](https://book.douban.com/subject/35136086/) - 2025.05.18 [[美] 海明威](https://www.douban.com/personage/27216434/)
<div>
	<img align="left" width="135" src='https://img2.doubanio.com/view/subject/s/public/s33692581.jpg'/><div>RATE: ★★★✰☆</div>
	<br/>
	<div>我的阅历有限，不确定海明威是否是多视角叙事的第一人，从效果而言，确实有一种独特的震撼。我最喜欢的是第一篇小说。我想这篇小说不如《乞力马扎罗的雪》盛名也许是因为它取了个十分老套的名字（玩笑）。当第一次读到，以被打猎的牛的视角来看待事物的时候，眼前是一亮的，毕竟没有人真的能为野牛设身处境着想。当一只野牛，接近死亡时会怎样，准备绝地反击时会怎样，充满愤怒情绪时会怎样——是的，生灵都有情绪——仿佛我们真的从他的文字里看到了一只现实里的濒临死亡的野牛，身临其境。《没有被斗败的人》里也是如此，我觉得斗牛这件事本身，很难写出一个长的、有意义的篇幅来，但他做到了。这些或许都建立在他亲身经历的基础上，不管是打猎或斗牛。至于《乞力马扎罗的雪》，无非又是意识流那一套，当然是顶级的意识流。我在很久之前读过，全然忘了。我甚至记得这本书的名字原先是《乞力马扎罗山的雪》，结果自然是我记错了，而且记错了很多年。没有“山”这个字，是有意为之的，他有着自己的考量，遵循他从简、极简的写作原则，同时想要凸显象征意义而非地理实体（AI 说的）。这件记错的小事没什么，但是对我自己来说是有趣的。
</div>
	<br/>
</div>

#### 5.[《在细雨中呼喊》](https://book.douban.com/subject/36028320/) - 2025.06.14 [余华](https://www.douban.com/personage/27208062/)
<div>
	<img align="left" width="135" src='https://img2.doubanio.com/view/subject/s/public/s34569411.jpg'/><div>RATE: ★★★✰☆</div>
	<br/>
	<div>不难看出，这部小说是带有一定自传性质的。里面或多或少都有作者往昔真实经历的回忆。他在自序里也写了，这是一部充满回忆的小说。有关叙事，他总是希望能够在一句完整的话里讲述完整的一件事、或者一个完整的段落，一个完整的篇章。整部小说的叙事，像是一个循环，或一个圆。你不知道是从哪里进入的，回过神的时候，已然身处故事之中，随着故事演进，时间的进程并不总是向前，回忆的片段并不总是往后，而到了最后，你发现自己又回到了原点，回到了这个圆的某处。我想这是这本书叙事的精妙之处。我能在阅读里，感受到作者对于汉语言文字运用的炉火纯青，甚至于炫技，也能隐隐感受到一些细微的不成熟。这些不成熟我说不上来，我只知道这本书是一个绝对华丽的开始，却不是他的巅峰。关于内容本身，我不喜欢故事里的主角总是那么苦了，我希望在结局他们能有幸福快乐的生活，即使那不真实，即使那很俗套。
